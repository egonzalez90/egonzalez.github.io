Desde su aparición en  2010, la comunidad de Openstack ha estado añadiendo nuevos componentes y funcionalidades.

A día de hoy estos son los componentes oficiales de OpenStack

Servicio Componente ** Descripcion**
Dashboard `Horizon <http://doc s.openstack.org/admi n-guide-cloud/conten t/ch_getting-started -with-openstack.html >`__ Interfaz gráfica que permite interactuar con Openstack para la administración básica del servicio
`Compute <http://doc s.openstack.org/admi n-guide-cloud/conten t/ch_getting-started -with-openstack.html >`__ Nova _ Encargada de la computación de la nube, gestiona y automatiza los recursos. Es la parte principal de OpenStack
Networking `Neutron <http://doc s.openstack.org/admi n-guide-cloud/conten t/ch_getting-started -with-openstack.html >`__ Gestiona la red y la asignación de IPs a las diferentes instancias
`Object Storage <http://docs .openstack.org/admin -guide-cloud/content /ch_getting-started- with-openstack.html> `__ Swift <http://docs. openstack.org/admin- guide-cloud/content/ ch_getting-started-w ith-openstack.html> __ Proporciona un Sistema de almacenamiento redundado y escalable. No es persistente el almacenamiento
`Block Storage <http://docs .openstack.org/admin -guide-cloud/content /ch_getting-started- with-openstack.html> `__ `Cinder <http://docs .openstack.org/admin -guide-cloud/content /ch_getting-started- with-openstack.html> `__ Proporciona un almacenamiento en bloque persistente
`Identity service <http://docs .openstack.org/admin -guide-cloud/content /ch_getting-started- with-openstack.html> `__ Keystone Servicio de autenticación y autorización de Openstack
`Image Service <http://docs .openstack.org/admin -guide-cloud/content /ch_getting-started- with-openstack.html> `__ `Glance <http://docs .openstack.org/admin -guide-cloud/content /ch_getting-started- with-openstack.html> `__ Openstack utiliza este servicio para el uso de las imagines de disco durante la asignación de instancias
Telemetry Ceilometer Monitorización para facturación con el cliente
Orchestration Heat _ Permite la orquestación de diferentes aplicaciones basadas en la nube, tales como AWS, etc.
Database  Trove <http://docs. openstack.org/admin- guide-cloud/content/ ch_getting-started-w ith-openstack.html> __ Una base de datos escalable para el uso de DBaaS
