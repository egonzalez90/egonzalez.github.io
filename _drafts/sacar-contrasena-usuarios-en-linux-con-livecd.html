Este es un tutorial bastante básico y fácil, no debería de darte ningún problema recuperar la contraseña de tu usuario de Linux.

Para realizar esta recuperación de password lo mas recomendable es usar un Live CD de alguna distribución como Backtrack, BackBox o Kali Linux, pero también podrías hacerlo con cualquier otro Linux, solo que tendrás que instalar John The Ripper, y las otras distribuciones ya vienen con el instalado.

El primer paso seria ir a esta ruta para hacerlo sobre el disco duro fisico que se quiera atacar.

cd /media/nombredeldiscoduro

Para saber cual es el disco duro lo mas sencillo es que cuando pones cd /media/ des doble tabulador y te mostrara las opciones que tienes a seleccionar

Una vez dentro del disco duro, deberás concatenar el archivo passwd con shadow,  esto se hace con el comando unshadow

unshadow ./etc/passwd ./etc/shadow > archivodeprueba

Una vez hecho esto sin que te de ningún error, se procederá a desencriptar la contraseña del usuario/os mediante el comando john

john archivodeprueba

Esto generara una carga de pruebas de contraseñas mediante fuerza bruta y podrás ver la evolución del desencriptado.

Este paso puede tardar mucho, todo depende de la fortaleza de la contraseña.

Una vez termina, te muestra la contraseña que estabas analizando

Como veis, es un proceso bastante sencillo.
