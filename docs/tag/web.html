Crear Imagen AMI con AWScli

Sun 12 October 2014 | tags: amazonAMIAWSawscliCLIcreate.imagederegister-imagedescribe-imagesimageimagesinstanciaservicesweb

Crear imagen AMI de una instancia

aws ec2 create-image --instance-id i-7ac29a38 --name "PruebaImagen" --description "ImagenPrueba" --no-reboot

Ver información de una imagen AMI

aws ec2 describe-images --image-id ami-defc5ea9

Des-registrar una imagen AMI (borrarla)

aws ec2 deregister-image --image-id ami-f4fc5e83
